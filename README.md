### 주요기능

- 검색엔진 최적화

  - 구글 / 네이버에서 LongForYou Studio 검색 가능

<br/>

- User

  - 예약 문의 메세지 전송
  - 반응형 (770px 미만에서 모바일 화면으로 전환)

<br/>

- Admin

  - 로그인
  - 쵤영 절차 / 문의 / 상품 설명 변경
  - 예약 메세지 확인
  - 이미지 추가 / 삭제
  - 메인 이미지 변경

<br/>
